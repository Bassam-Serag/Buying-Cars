# Buying cars
This project is a comprehensive solution for buying cars,
developed using C# WinForms and CodeFirst approach for efficient database management.
With a user-friendly interface and robust backend, this application streamlines the car purchasing process.

![image](https://github.com/Bassam-Serag/Buying-Cars/assets/105117034/7fe0f189-4751-4a0b-b2ab-4fdfca36cc7d)


### Prerequisites

Before you begin, ensure you have met the following requirements:

* [Git](https://git-scm.com/downloads "Download Git") must be installed on your operating system.


### Run Locally

To run **Buying cars** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Bassam-Serag/Buying-Cars.git
```

Windows:

```bash
git clone https://github.com/Bassam-Serag/Buying-Cars.git
```
